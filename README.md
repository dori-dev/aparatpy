# [APARATPY](https://pypi.org/project/aparatpy/)

aparatpy is a video downloader library for [aparat.com](https://aparat.com)

#

## Install Aparatpy

```
pip install aparatpy
```

#

## Use Aparatpy

```python
from aparatpy import Main

video = Main('video-url', 'video-quality')
video.download()
```
